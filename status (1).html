Documentation API de paiement

Cette API permet de récupérer l'état d'une transaction en utilisant sa référence.

Exemple de réponse :

200 OK

{
    "marchant_id":"PP-F1096",
    "pay_id":"CI2023013113165473",
    "reference":"123456789",
    "amount":1000,
    "channel":"CARD2",
    "token":"af8066cdac5e35d695dfd66ba12a63d8",
    "date":"2023-01-31 13:20:51",
    "success":true
}

marchant_id Identifiant marchand
pay_id Identifiant de paiement de l'opérateur
reference Référence du marchand
amount Montant
channel Mode de paiement
token Token
date Date de la transaction
success Statut de la reponse

Utilisation de l'API :

Envoyez une requête GET à l'URL suivante :

https://api.paiementpro.net/status/{reference}

Remplacez {reference} par la référence de la transaction.
